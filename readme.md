# Para execução do código pela primeira vez, seguir a ordem abaixo:

* Ordem de execução do sistema pra criar o recomendador 
1 - Executar "basededados.ipynb"
2 - Executar "dados_sinteticos.ipynb"
3 - Verificar se foi criado database "biblioteca"
4 - Realizados os passos anteriores, o recomendador nas versões para console ("recomendador.ipynb") e front-end ("app.py") já estarão prontos para rodar.

*Para rodar o recomendador no Flask, digitar no Terminal PS:
- cd projetos (para entrar na pasta correta) 
- python app.py (para executar o programa e gerar o servidor local)

* Ao realizar os passos acima, o arquivo "eda.ipynb" poderá ser executado e exibirá os gráficos atualizados.

* Para rodar o recomendador na base de validação:
- Executar todos os blocos do "basevalidacao.ipynb" (desde a criação da tabela sintética até o sistema pedir entrada de usuário com os valores de teste da tabela verdade).
